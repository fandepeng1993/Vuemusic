个人花了几个小时候时间经过测试完成对于git项目开发gitHUB部署仓库常见命名做了总结：
对于远程分支和本地分支建立，删除，合并，跟踪，管理这块都做出了总结命令：

git add file-name 添加修改文件

git status 查看本分支状态

git pull 拉取代码

git clone url 克隆代码

git push 推送代码到GIT服务端远程分支

git branch -a 查看所有分支
git remote -v 查看所有仓库

git checkout branch-name 切换本地分支
git checkout -b branch-name 本地建立远程分支并切换到本地分支

git push --set-upstream origin branch-name 本地分支与远程分支建立连接，跟踪远程分支


git branch -r -d origin/branch-name 删除远程分支跟踪器
git push origin :branch-name 删除远程分支并且删除跟踪器

git branch -D branch-name 删除本地分支

git merge branch-name 与该本地分支合并
